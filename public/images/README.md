## ğŸ¨ How to Add Screenshots

### Option 1: Actual Screenshots
1. Take screenshot dari project lu yang udah deployed
2. Crop ke ukuran 1200x630px
3. Optimize dengan TinyPNG atau ImageOptim
4. Rename sesuai nama di atas
5. Copy ke folder ini

### Option 2: Mockup/Design
1. Bikin mockup pakai Figma/Canva
2. Export sebagai PNG
3. Rename dan copy ke folder ini

### Option 3: Placeholder (Temporary)
Kalau belum ada screenshot, portfolio akan show placeholder:
"Screenshot Coming Soon" dengan icon
