## ğŸ› TROUBLESHOOTING

**Images not showing?**
- Check file names (case-sensitive!)
- Check files in `public/images/`
- Clear browser cache
- Restart dev server

**Build errors?**
- Delete `node_modules`
- Run `npm install` again
- Check Node.js version (need 18+)

**Styling issues?**
- Run `npm run dev` fresh
- Check Tailwind classes
- Inspect in browser DevTools
