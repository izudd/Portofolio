## ‚ùì FAQ

**Q: Gua belum punya screenshot, gimana?**
A: No problem! Portfolio akan show placeholder "Screenshot Coming Soon". Bisa ditambahin nanti.

**Q: Image gua terlalu gede (5MB+), gimana?**
A: Compress pakai TinyPNG.com. Target: <500KB per file.

**Q: Format apa yang didukung?**
A: PNG, JPG, atau SVG. PNG recommended untuk quality.

**Q: Ukuran harus EXACT 1200x630px?**
A: Ngga harus! Tapi aspect ratio 16:9 lebih bagus.

**Q: Bisa pakai mockup instead of screenshot?**
A: YES! Malah lebih keren kalau pake mockup yang professional.

**Q: File name case-sensitive?**
A: YES! Harus lowercase exact: `pam-jaya.png` bukan `PAM-JAYA.PNG`

## üí° Pro Tips

1. **Use Incognito Mode** - Screenshot tanpa extension/bookmark bar
2. **Fullscreen Browser** - Hide address bar untuk clean look
3. **Light Mode** - Kalau project lu ada dark mode, screenshot light mode lebih jelas
4. **Hide Personal Data** - Blur nama, email, data sensitive
5. **Consistent Style** - Semua screenshot dari angle/zoom yang sama
